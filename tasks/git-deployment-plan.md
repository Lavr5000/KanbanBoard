## Описание проекта:

Это Kanban Board 2.0 - совершенно новая версия приложения, разработанная с нуля.
Переписана вся кодовая база с использованием современных технологий и лучших практик.

### Основные особенности:
- Next.js 15 с App Router
- TypeScript
- Tailwind CSS
- Zustand для управления состоянием
- Премиальная темная тема
- Расширенная система приоритетов (4 уровня)
- Продвинутая система дедлайнов с визуальными индикаторами
- Drag & Drop функциональность

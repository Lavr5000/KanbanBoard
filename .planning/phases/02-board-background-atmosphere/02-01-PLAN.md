# 02-01: Gradient Mesh and Noise Texture

## Objective

Create the foundational background layers: an animated gradient mesh and a noise texture overlay. These provide the atmospheric depth and visual richness for the board.

**Purpose:** Replace the plain `#121218` background with a living, breathing atmosphere that enhances the glassmorphism design.

**Output:** Two reusable background components with GPU-optimized animations.

## Context

@tailwind.config.ts (has `gradient-shift` animation configured)
@src/app/globals.css (plain `#121218` background on body)

**File 1: AnimatedGradient.tsx**
```tsx
"use client";

import React from "react";

/**
 * Animated gradient mesh background.
 * Uses 400% background size for smooth gradient shifting animation.
 * Purple-to-blue theme matches glassmorphism design.
 */
export function AnimatedGradient() {
  return (
    <div className="fixed inset-0 -z-20 overflow-hidden">
      {/* Base gradient layer - static depth */}
      <div className="absolute inset-0 bg-gradient-to-br from-purple-900/40 via-blue-900/40 to-slate-900/40" />

      {/* Animated gradient layer - 30s cycle, 400% size for smooth movement */}
      <div className="absolute inset-0 animate-gradient-shift bg-gradient-to-br from-purple-500/20 via-blue-500/20 to-pink-500/20 bg-[length:400%_400%]" />
    </div>
  );
}
```

**File 2: NoiseTexture.tsx**
```tsx
"use client";

import React from "react";

/**
 * SVG noise texture overlay for subtle grain effect.
 * Uses base64 SVG filter with fractalNoise.
 * Very low opacity (0.03) for subtle texture without distraction.
 */
const noiseSvg =
  "data:image/svg+xml,%3Csvg viewBox='0 0 256 256' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='noise'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.9' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23noise)'/%3E%3C/svg%3E";

export function NoiseTexture() {
  return (
    <div
      className="fixed inset-0 pointer-events-none -z-10 opacity-[0.03]"
      style={{
        backgroundImage: `url('${noiseSvg}')`,
        backgroundRepeat: 'repeat',
      }}
    />
  );
}
```

**File 3: index.ts**
```tsx
export { AnimatedGradient } from './AnimatedGradient';
export { NoiseTexture } from './NoiseTexture';
```

Key implementation notes:
- AnimatedGradient uses `-z-20` (behind everything)
- NoiseTexture uses `-z-10` (above gradient, below content)
- `animate-gradient-shift` is already in tailwind.config.ts
- Both use `fixed inset-0` for full-screen coverage
- `pointer-events-none` on noise for click-through

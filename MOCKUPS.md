## ТЕКУЩЕЕ СОСТОЯНИЕ (ДО)

### Текущая карточка
```
┌─────────────────────────────────────┐
│ #a2b3c (ID)         [Grip] [Delete] │ ← Скрыто до hover
│                                     │
│ ○ ●  Новая задача                   │ ← Цветная точка (приоритет)
│ Введите описание...                 │ ← Часто пусто/шум
│                                     │
│ medium                              │ ← Priority badge
│ (Дополнительный шум)                │
└─────────────────────────────────────┘

При клике для редактирования:
┌─────────────────────────────────────┐
│ [INPUT] Новая задача                │ ← Редактируемое поле
│ [TEXTAREA] Введите описание... (60px)│ ← Часто пусто
└─────────────────────────────────────┘
```

**Проблемы:**
- 3-4 строки текста, но только 2 редактируемых
- ID и priority badge не используются
- Много визуального шума
- 60% пространства карточки неиспользуемо (пустое описание)
